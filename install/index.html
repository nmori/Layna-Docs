    インストール¶

    • サイトからダウンロードしたプログラムを開きます。

    • インストールがおわるとスタートメニューに項目が増えます。使う方のメニューを起動してください。

    起動しない場合について

    • .Net4.8ランタイムが必要です。.Net4.8ランタイムを導入してください。
